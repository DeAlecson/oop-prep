# QuizApp

QuizApp is designed to help with on‑the‑go Python OOP and fundamental syntax quizzes.  
If you have an AI API key, an integrated AI marking feature can automatically grade your assessments.  

This project is meant for **personal use**, but you’re welcome to fork it into your own projects.

---

## Features
- Practice quizzes on Python OOP and syntax
- AI‑powered marking (optional, requires API key)
- Lightweight design for quick study sessions
